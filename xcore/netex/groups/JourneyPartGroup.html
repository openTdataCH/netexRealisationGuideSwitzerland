233.8. The group netex:JourneyPartGroup

netex:JourneyPartGroup

Elements for JOURNEY PART.
netex:JourneyPartGroup

netex:Description

0:1

+netex:MultilingualString

Description of JOURNEY PART.

netex:ParentJourneyRef

0:1

>netex:VehicleJourneyRefStructure

Reference to parent of which this is part. If given by context does not need to be stated.

netex:DayOffsetRelativeToMainPart

0:1

netex:DayOffsetType

OPERATING DAY day offset of parent journey relative to OPERATING DAY date of main JOURNEY PART journey. Positive value indicates a later OPERATING DAY date for parent journey. Does not need to be stated if 0 or not relevant for context.
netex:JourneyPartReferencesGroup

0:1

>netex:JourneyPartRefStructure

Main JOURNEY PART for journey.

→‍netex:TrainNumberRef

0:1

>netex:TrainNumberRefStructure

Reference to a TRAIN NUMBER.

→‍netex:BlockPartRef

0:1

>netex:BlockPartRefStructure

Reference to a BLOCK PART.

The element contains none or one of the following element sequences (a-b)

a

netex:FromStopPointRef

-1:1

>netex:ScheduledStopPointRefStructure

Point at which this JOURNEY PART starts.

a

netex:ToStopPointRef

-0:1

>netex:ScheduledStopPointRefStructure

Point at which this JOURNEY PART ends. If not set it is until the end of the JOURNEY. We will make this element mandatory in the future.

b

netex:FromPointInPatternRef

-1:1

>netex:PointInSequenceRefStructure

POINT at which applicability of JOURNEY PART starts. When using PASSING TIME this element is to be used. +v2.1

b

netex:ToPointInPatternRef

-1:1

>netex:PointInSequenceRefStructure

POINT at which applicability of JOURNEY PART ends. +v2.1
