# TransportMode
NeTEx definies the Physical Model for reusable transport mode normally implements the TRANSPORT MODE and SUB-MODE as enumerated values so that the value used are standardised.

| trolleyBus | 	Trolley Bus  |
| tram | 	Tram  |
| water | 	Water  |
| cableway | 	Cableway  |
| other | 	Other mode  |
