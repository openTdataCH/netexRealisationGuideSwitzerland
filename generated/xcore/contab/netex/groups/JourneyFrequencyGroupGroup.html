262.6. The group netex:JourneyFrequencyGroupGroup

netex:JourneyFrequencyGroupGroup

Elements for a JOURNEY FREQUENCY GROUP.

The element contains one of the following element sequences (a-b)
netex:JourneyFrequencyGroupGroup

a

netex:FirstDepartureTime

-1:1

Time of first departure in JOURNEY FREQUENCY GROUP at the first stop.

a

netex:FirstDayOffset

-0:1

netex:DayOffsetType

Offset days for the first departure time. Number of days after the starting operational day if journey if not on the same calendar day. Default is 0 for the same day.

a

netex:LastDepartureTime

-0:1

Time of last departure in JOURNEY FREQUENCY GROUP.

a

netex:LastDayOffset

-0:1

netex:DayOffsetType

Offset days for the last departure time. Number of days after the last departure time of the journey if not same calendar day. Default is 0 for the same day.

b

netex:FirstArrivalTime

-1:1

Time of first arrival in JOURNEY FREQUENCY GROUP at the last stop.

b

netex:FirstDayOffset

-0:1

netex:DayOffsetType

Offset days for the first arrival time. Number of days after the first arrival time of the journey if not same calendar day. Default is 0 for the same day.

b

netex:LastArrivalTime

-0:1

Time of last arrival in JOURNEY FREQUENCY GROUP.

b

netex:LastDayOffset

-0:1

netex:DayOffsetType

Offset days for the last arrival time. Number of days after the last arrival time of the journey if not same calendar day. Default is 0 for the same day.

netex:timeDemandTypes

0:1

+netex:timeDemandTypeRefs_RelStructure

TIME DEMAND TYPES associated with JOURNEY FREQUENCY GROUP.

netex:journeys

0:1

+netex:explicitJourneyRefs_RelStructure

TIME DEMAND TYPES associated with JOURNEY FREQUENCY GROUP.
