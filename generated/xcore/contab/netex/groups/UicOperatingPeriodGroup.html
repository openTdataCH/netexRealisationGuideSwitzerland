112.15. The group netex:UicOperatingPeriodGroup

netex:UicOperatingPeriodGroup

Elements for a UIC OPERATING Period.
netex:UicOperatingPeriodGroup

netex:ValidDayBits

1:1

String of bits, one for each day in period: whether valid or not valid on the day. Normally there will be a bit for every day between start and end date. If bit is missing, assume available.

netex:DaysOfWeek

0:1

netex:DaysOfWeekListOfEnumerations

Days of week to which bits correspond.
