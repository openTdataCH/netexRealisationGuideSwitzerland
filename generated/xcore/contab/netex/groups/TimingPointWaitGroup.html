208.9. The group netex:TimingPointWaitGroup

netex:TimingPointWaitGroup

Elements for a TIMING POINT IN JOURNEY PATTERN.
netex:TimingPointWaitGroup

netex:IsWaitPoint

0:1

Whether point is a wait point.

The element contains none or one of the following elements (a-b)

a

netex:WaitTime

-1:1

Wait time as interval. OPTIMISATION assuming default Time Demand. Use this to declare a single wait time Other wise a list of times for different time demands.

b

netex:waitTimes

-1:1

+netex:journeyPatternWaitTimes_RelStructure

Wait times for (TIMING) POINT IN JOURNEY PATTERN. There may be different times for different time demands.

netex:headways

0:1

+netex:journeyPatternHeadways_RelStructure

Set of HEADWAYs (frequency of services) that applies at a (TIMING) POINT IN JOURNEY PATTERN.
