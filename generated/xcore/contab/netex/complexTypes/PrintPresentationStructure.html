137.11. The complex type netex:PrintPresentationStructure

netex:PrintPresentationStructure

Types describing common presentation properties for Print.

netex:Colour

0:1

netex:PrintColourValueType

Default colour value for printed graphics associated with ENTITY - e.g. of a LINE. This may be, e.g., a CMYK or RAL code, not intended to be machine-readable.

netex:ColourName

0:1

Name of default colour value for printed graphics associated with entity., eg Pentone name

netex:ColourSystem

0:1

Name of colour system used for printed ColourName and TextColourName: for example, - RAL, https://en.wikipedia.org/wiki/RAL_colour_standard; - DIN 6164 http://www.dtpstudio.de/atlas/farbsysteme/DIN%206164_bs00_3.htm; - Pantone (be aware that Pantone is proprietary), etc.

netex:BackgroundColour

0:1

netex:PrintColourValueType

Default background colour value for printed text and graphics associated with ENTITY. This may be, e.g., a CMYK or RAL code, not intended to be machine-readable.

netex:BackgroundColourName

0:1

Name of the backgropund colour value (in ColourSystem) for printed text and graphics associated with entity.

netex:TextColour

0:1

netex:PrintColourValueType

Default colour value for printed text associated with entity. This may be, e.g., a CMYK or RAL code, not intended to be machine-readable.

netex:TextColourName

0:1

Name of default colour value for printed text associated with entity, in specified ColourSystem.

netex:TextFont

0:1

Identifier of font for printed text.

netex:TextFontName

0:1

Name of font for printed text.

netex:TextLanguage

0:1

Default language for printed text.

netex:FontSize

0:1

netex:FontSizeEnumeration

Default language for printed text +v1.1.
