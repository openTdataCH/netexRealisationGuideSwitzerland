47.3. The complex type netex:CodespaceStructure

netex:CodespaceStructure

netex:EntityStructure

Type for ENTITY.

The element content starts with items inherited from the base type:

@nameOfClass

1:1

netex:NameOfClass

Name of Class of the ENTITY. Allows reflection. Fixed for each ENTITY type.

@id

1:1

netex:ObjectIdType

Identifier of ENTITY.

Inherited content is followed by own content:

@dataSourceRef

1:1

netex:DataSourceIdType

Data Source of ENTITY.

netex:Xmlns

1:1

netex:CodespaceIdType

Prefix used to identify CODESPACE, e.g. 'napt', 'oda', 'ratp' etc.

netex:XmlnsUrl

0:1

CODESPACE path. The URL associated which with the CODESPACE defines a unique system for identifying objects within the CODESPACE.

netex:Description

0:1

+netex:MultilingualString

Description of CODESPACE.
