# Content of a Swiss zip file
This folder shows the content of a Swiss export zip with the files that are used. We only use a few lines and withion those only 2.3 `ServiceJourney`.
We also remove some intermediate stops
We use the files structure as defined [here](../../docs/04_files.md)

It contains:
- Partial modeling of "Wankdorf"
- train line A: Operat or SBB, Bern - Wankdorf  Olten
- train line B: Operator BLS, Bern - Wankdorf - Biel
- train line: - tram line C: Operator Bernmobil, Tram No 9 to Wankdorf, Bahnhof - Guisanplatz - Wabern
- bus line D: RBS Breitfeldplatz - Wankdorf Bahnhof Nord - Münchenbuchsee
- interchanges
